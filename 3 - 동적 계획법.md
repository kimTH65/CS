<h6>피보나치 수는 나누어지지 않는 수를 말하며, 대표적인 동적계획법 문제이다.<br>
(フィボナッチ数は割れない数をいい、代表的な動的計画法問題である)
<br><br>
f(0) = 1 , f(1) = 1 일 때 n>1 인 경우 f(n) = f(n-1) + f(n-2) 로 구현 가능하다.<br>
(f(0) = 1、f(1) = 1のとき n> 1 の場合 f(n) = f(n-1) + f(n-2) で具現可能である)
</h6>  
<br>

```python

def fib(n):
    if n==1 or n==2:
        return 1
    else:
        return fib(n-1) + fib(n-2)


```
